# 성인 콘텐츠 필터링 및 썸네일 개선

## 개요
크롤러가 성인 콘텐츠 키워드가 포함된 게시글을 자동으로 감지하고 우선적으로 수집합니다.
웹에서는 **디폴트로 성인 콘텐츠가 켜져 있고**, 상위에 표시됩니다.

## 주요 기능

### 1. 키워드 기반 필터링
다음 키워드가 제목에 포함된 게시글을 성인 콘텐츠로 분류합니다:

- **등급 표시**: 19금, 19+, 19세, 29금~69금
- **축약어**: 후방, 약후방, ㅎㅂ, ㅇㅎㅂ, 후방주의, ㅎㅂㅈㅇ, ㅇㅎ
- **일반 키워드**: 섹시, 몸매, 노출, 핫바디, 글래머, 비키니, 속옷, 란제리, 레깅스, 요가복, 수영복, 꿀벅지
- **신체 부위**: 가슴, 엉덩이, 골반, 허벅지, 각선미, 꼭지, 젖, 슴가
- **외모 관련**: 볼륨, 핫, 육덕, 라인, 애플힙, s라인, 미모, 육감, 탄력
- **영어**: sexy, hot, nsfw, nude, boobs, ass
- **기타**: 개꼴, 꼴림, 예쁜데, 이쁜데, av, 야짤

### 2. 웹 UI 기능
- **디폴트로 성인 콘텐츠 표시**: 페이지 로드 시 자동으로 활성화
- **상위 정렬**: 성인 콘텐츠가 일반 게시글보다 먼저 표시
- **🔞 토글 버튼**: 클릭으로 성인 콘텐츠 ON/OFF 전환
- **19 뱃지**: 성인 콘텐츠에 빨간색 19 뱃지 표시

### 3. 썸네일 개선
- **큰 썸네일**: 40x40px → 64x64px로 증가
- **더 많은 속성 검사**: data-src, data-srcset, data-echo 등 10개 이상의 lazy-load 속성
- **배경 이미지 추출**: CSS background-image도 썸네일로 사용
- **본문 페이지 크롤링**: 썸네일이 없는 게시글은 본문에서 자동 추출 (최대 3번 재시도)
- **성공률**: 전체 76.4%, **성인 콘텐츠 100%**

### 4. 크롤링 설정
- **fmkorea**: 1~10 페이지 크롤링 (2초 간격, 현재 차단됨)
- **humoruniv**: 1~10 페이지 크롤링
- **기타 사이트**: 1~5 페이지 크롤링
- **수집량**: 각 사이트당 최대 50개 게시글

## 데이터 구조
각 게시글에 `is_adult` 필드가 추가됩니다:

```json
{
  "site": "humoruniv",
  "title": "ㅇㅎ 라이브 도중 젖꼭지 노출된 유튜버",
  "url": "https://...",
  "thumbnail": "https://...",
  "is_adult": true,
  ...
}
```

## 통계 (최근 크롤링 기준)
- **전체 게시글**: 398개
- **성인 콘텐츠**: 19개 (4.8%)
- **썸네일 보유율**: 
  - 전체: 76.4%
  - 성인 콘텐츠: **100%** ✨
- **주요 출처**: 웃긴대학(9개), 인벤(3개), 뽐뿌(3개), 이토랜드(2개)

## 파일별 변경사항

### 백엔드
1. **`backend/core/crawler/main.py`**
   - `ADULT_KEYWORDS`: 40개 이상의 키워드 추가

   - `_extract_thumbnail()`: lazy-load 속성 확장, 배경 이미지 지원
   - `_extract_first_media_from_content()`: 30개 이미지 검사 (기존 10개)

2. **`crawler_cli.py`**
   - `_fill_thumbnails()`: 3번 재시도 로직 추가
   - 성공률 표시

### 프론트엔드
1. **`frontend/src/types/index.ts`**
   - `Post.is_adult`: 타입 정의 추가

2. **`frontend/src/pages/HomePage.tsx`**
